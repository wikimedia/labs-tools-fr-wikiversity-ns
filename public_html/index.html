fr-wikiversity-ns is a tool for French Wikiversity mains customs namespaces monitoring.
Basicly two simples Python scripts based on Pywikibot:
# fac.py is scanning the faculty namespace (106)
# dpt.py is scanning the department namespace (108)

Each script is collecting a set of data and save it in a Lua module table, respectively Module:Table faculté and Module:Table département.
In a second step some functions of modules Faculté vues and Département vues will print back datas for dashbords, charts, and globaly monitoring schools (Facultés) and departments (Départements).

For more informations see the Wikiversité laboratory experiences linked below.

fr-Wikiversity-ns est un outil pour les espaces de noms supplémentaires du projet Wikiversité francophone.
Initialement deux scripts Python basés sur Pywikibot:
# fac.py parcours l'espace de noms Faculté (106)
# dpt.py parcours l'espace de noms Département (108)

Chaque script collecte un jeu de données et le sauvegarde dans une table via un module Lua ; respectivement Module:Table faculté et Module:Table département.

Dans un deuxième temps, les fonctions des modules Faculté vues et Département vues restitueront les données sous forme de tableaux et classements, synthétiques ; capables de résumer les informations utiles, relatives aux espaces de noms "Faculté" et "Département".

Pour plus d'informations voir les expériences du laboratoire ci-dessous.
